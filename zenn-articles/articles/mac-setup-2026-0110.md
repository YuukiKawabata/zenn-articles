## 📋 システム設定（操作の摩擦をゼロにする）

まずは「全Macユーザーがやるべき」レベルの基本設定から。操作の無駄を削ぎ落とし、集中できる環境を作ります。

### トラックパッド・キーボードの最適化

- **トラックパッド設定**
  - **タップでクリック**を有効化（押し込む力すら節約）。
  - **3本指でドラッグ**を有効化（アクセシビリティから設定。これが最強。押し込む力すら節約）。
  - **軌跡の速さ**を最大に変更。（最小の動作で画面の端から端まで移動できる）
- **キーボード設定**
  - **キーのリピート**：速く
  - **リピート入力認識までの時間**：短く

### 画面とFinderの整理

- **デスクトップとDock**
  - **サイズ**を小さく調整し、**「Dockを自動的に表示/非表示」**をオン。
  - **配置**は画面領域を有効活用するために左側も検討。
- **Finder設定**
  - **すべてのファイル名拡張子を表示**（エンジニアの必須設定）。
  - **パスバー・ステータスバーを表示**。
  - 隠しファイルの表示切り替え（`Command + Shift + .`）を覚えておく。
- **外観と壁紙**
  - 落ち着く**ダークモード**に設定。
  - 壁紙は暗めのシンプルなものをチョイス。

## 🛠 開発環境（モダンな開発スタック）

### パッケージ・ランタイム管理

- **Homebrew**：これがないと始まらない。
- **mise** ：**【2026年新定番？】** Node.js, Python, Rubyなどをこれ1つで管理。とにかく爆速みたい。
- **nvm (Node.js)**：プロジェクト指定がある場合に使用。
- **pyenv (Python)**：Python環境の切り替え用。
- **Docker Desktop**：ローカルを汚さない開発環境の要。

### Git設定

- **Git**：最新版をインストール。
- **Git基本設定**：Bash

```git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
git config --global init.defaultBranch main
git config --global core.editor "code --wait"
```

### キーボード・周辺機器最適化

- **Karabiner-Elements**

  - MacとWindowsの打鍵感を共通化するためのこだわり設定。
- **Logicool Options+**
  - マウスのボタンカスタマイズ用。
